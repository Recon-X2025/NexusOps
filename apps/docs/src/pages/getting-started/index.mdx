# Getting Started

This guide walks you through running NexusOps locally for development or evaluation purposes.

## Prerequisites

Before you begin, ensure you have the following installed:

| Tool | Minimum Version | Notes |
|------|----------------|-------|
| Node.js | 18.x LTS or 20.x | Required by Next.js 15 |
| pnpm | 8.x or 9.x | Workspace package manager |
| PostgreSQL | 15.x | Primary datastore |
| Redis | 7.x | Session store and job queue |
| Git | 2.x | Source control |

> **macOS users**: Install PostgreSQL and Redis via [Homebrew](https://brew.sh): `brew install postgresql@15 redis`.
>
> **Windows users**: Use WSL2 with Ubuntu, or Docker Desktop.

## 1. Clone the Repository

```bash
git clone https://github.com/Recon-X2025/NexusOps.git
cd NexusOps
```

## 2. Install Dependencies

NexusOps uses [pnpm workspaces](https://pnpm.io/workspaces). Install all workspace dependencies with a single command:

```bash
pnpm install
```

This installs dependencies for all apps and packages in the monorepo simultaneously.

## 3. Set Up Environment Variables

Each application has its own `.env` file. Start by copying the example files:

```bash
cp apps/api/.env.example apps/api/.env
cp apps/web/.env.example apps/web/.env
```

Open `apps/api/.env` and fill in the required values. At minimum you need:

```env
DATABASE_URL="postgresql://postgres:password@localhost:5432/nexusops"
REDIS_URL="redis://localhost:6379"
JWT_SECRET="change-me-to-a-long-random-string"
APP_SECRET="change-me-to-another-long-random-string"
```

See the [Configuration reference](/getting-started/configuration) for the full list of environment variables.

## 4. Set Up the Database

Create the PostgreSQL database, then run migrations:

```bash
createdb nexusops
pnpm --filter @nexusops/db db:migrate
```

Optionally, seed the database with demo data including sample organizations, users, and tickets:

```bash
pnpm --filter @nexusops/db db:seed
```

## 5. Start the Development Servers

Run all services in parallel using the root-level dev script:

```bash
pnpm dev
```

This starts:

| Service | URL | Description |
|---------|-----|-------------|
| Web (Next.js) | http://localhost:3000 | Main application frontend |
| API (tRPC) | http://localhost:3001 | Backend API server |
| Docs (Nextra) | http://localhost:3003 | This documentation site |

## 6. Log In

Navigate to [http://localhost:3000](http://localhost:3000). If you ran the seed script, use:

- **Email**: `admin@nexusops.local`
- **Password**: `admin123`

> Change the default admin password immediately after first login in **Settings → Security**.
